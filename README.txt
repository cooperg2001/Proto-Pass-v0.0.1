Proto Pass v0.0.2 is a prototype open source password
program meant to be integrated with other programs.

NOTE: This program is written in C++ using codeblocks IDE
NOTE: Files include full codeblocks project
NOTE: Version numbering works as follows (/'s represent .) Major/Minor/Build.
